Contributing To Bitcoin Core

Thank you for considering contributing to Bitcoin Core! We have instructions below to help you get started contributing in several different ways.

If want to contribute in another way, please visit the #bitcoin-dev IRC chatroom and discuss your plan with a developer.

  • Bug reports
    Report bugs, including security issues

  • Code
    Write & review code

  • Documentation
    Write documentation for users and developers

  • Translations
    Translate the user interface

  • Tech support
